Demonstrando a instalação e uso do Mensagens Rápidas Interaxa.

Depois de baixar, extraia o arquivo.

Sugiro favoritar o arquivo e deixar na Barra de Favoritos.

Clique no botão "Editar nome/atendente" para incluir o seu nome.

Importante: Clique em "Receber avisos de alterações" para ser notificado(a) de mudanças no conteúdo.

Clique para selecionar a mensagem. Ctrl + C para copiar.

Cole a mensagem no Interaxa e envie para o cliente.
