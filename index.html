Demonstrando a instalação e uso.

Usando as mensagens

Clique no botão "Editar nome/atendente" para incluir o seu nome. Não é possível incluir o nome do cliente.

Seja avisado(a) de alterações no conteúdo.

Basta clicar no botão "Receber avisos de alterações".

Clique para selecionar a mensagem. Ctrl + C para copiar.

Cole a mensagem no Interaxa e envie para o cliente.
